Missing Features I wanted to add, but did not have the data to support:

Trend analysis of each Repos historical data, such as:
  1. Stars, Watchers, Downloads growth over time
  2. Graphs to compare over all growth rate
  3. Graphs to visualize the velocity of change of each repo over time
  4. Metric Ratios such as Stars to age(this I actually do have the data for)
  5. Open issues over time, and rate of change
Other cool but out of scope ideas:
  1. Analyze active public repos using each framework and it's changes over time, and possibly adding a regression line to crudely predict the "future"
  2. Web crawl certain sites and run comments a sentiment analysis on each framework.(I am currently working on a project that is loosely similar)
